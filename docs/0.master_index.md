## ğŸ”§ Troubleshooting Guide

### Common Issues by Step

**Step 1: Directory Creation**

- Issue: Permission denied
- Solution: Check directory ownership, use `sudo` if needed

**Step 2: Dependency Installation**

- Issue: "No module named X"
- Solution: Activate venv, reinstall dependencies

**Step 3: Docker Build**

- Issue: "Dockerfile not found"
- Solution: Ensure correct path, run from project root

**Step 4: GitHub Actions**

- Issue: Workflow doesn't run
- Solution: Check syntax, ensure `.yml` file format

**Step 5: API Errors**

- Issue: 502 Bad Gateway
- Solution: Check if all services are running, review logs

See individual step documentation for detailed troubleshooting.
